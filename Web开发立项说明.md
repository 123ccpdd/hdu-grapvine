# HDU grapevine

## 网站概述：
网站主要有以下几个设想:  
1. 推送跟计算机学院有关的各种最新通知  
2. 添加计算机考研课程的各种资源（例如题目，知识点讲解，可讨论的个人总结）  
3. 激励同学们学习的文章或者一些知识性的文章推送（也称心灵鸡汤）

## 个人特色：    
本人在校期间经常需要在qq群等渠道获取学校通知，而这些通知往往是临时性质的，而且往往只有一两天的有效期，所以需要一个能够实时更新的网站来满足需求。并且博主或者管理员可以更新时事新闻增强同学们对于热点新闻和时政的关注。同时考研的同学也需要一个能够满足学习需求的网站来提高自身的学习效率并及时进行交流和讨论。

## 内容类型：文章、评论等  
- 内容类型包括：开发者新闻、通知、心灵鸡汤、考研课程资源等
- 新闻：本来是打算做新闻联播之类，但是太过枯燥。打算提托开发者进行，分为前端，后端，python,java的一些在知名开发者论坛的文章，然后再进行整合，形成一个完整的新闻系统。
- 通知：包括紧急通知，通告公告，校园新闻公告，活动通知。 
- 心灵鸡汤：包含各种刊物的写的非常好的文章。 
- 考研课程资源：包括考研课程资源、考研题目、考研知识点讲解、考研个人总结等。

## 页面类型：  
### 站点首页 home
站点入口，由以下几个部分组成：
- 导航栏：包括首页，新闻，通知，心灵鸡汤，考研课程资源等
- 信息区块：重点突出最近新发布的或者最火的信息列表和摘要
- 广告区块（到时候看，主打一个兴趣）
- 登录界面：提供注册和登录功能  
### 内容栏目首页 index 
主要由以下几个栏目的首页：新闻，通知，心灵鸡汤，考研课程资源。
### 内容详细页 show
站点首页或者内容栏目首页所提供的链接点进去的内容，主要是新闻，通知，心灵鸡汤，考研课程资源等。
注意，页面附带评论功能，点赞功能，转发功能（多半做不到）

## 角色类型：  
- 访客：只能读取公共内容，可评论（未登录访客不可评论）  
- 站长或管理员：所有权限，可以任意输入和编辑内容，以及删除不良评论

### 前台功能  
新闻展示：提供新闻内容的展示，包括标题、摘要、正文、图片、视频等。新闻可以按时间顺序排列或通过分类、标签进行筛选和导航。  
导航和搜索：提供导航菜单或标签，让用户快速浏览和定位到感兴趣的新闻栏目或主题。同时，提供搜索框让用户可以输入关键词搜索相关的新闻内容。  
分类和标签：将新闻内容按照不同的分类进行组织，如重要新闻、重要通知、临时通知、文章等。同时，为每篇文章添加标签或关键词，以方便用户根据喜好或主题进行浏览。  
评论和互动：允许注册用户或访客对新闻进行评论（点赞或分享等互动操作待定），以促进用户参与和社交互动。  
用户注册和登录：为用户提供注册账号和登录功能。用户登录后才允许浏览重要内容。  
### 后台功能  
文章管理：包括发布新闻、编辑新闻、删除新闻等操作。后台管理员可以通过所见即所得的编辑器来创建新闻内容，同时可以设置发布时间、作者、标签、分类等属性。  
栏目管理：允许管理员创建、编辑和删除新闻栏目或分类，以便对新闻内容进行组织和分类。  
广告管理：提供管理员管理网站广告的功能，包括新增广告位、上传广告素材、设置广告展示时间和位置等。  
用户管理：后台管理员可以管理用户账号，包括用户注册审核、用户信息修改、用户权限管理等。此外，还可以设定用户角色，以便区分管理员普通用户的权限。  
评论管理：允许管理员对新闻评论进行审核、回复和删除等操作，以维护网站的秩序和内容质量。  
搜索引擎优化管理：提供后台设置网站的元标签、关键词、网站地图等功能，以提升网站在搜索引擎中的可见性和排名。（待定）  
系统设置：提供网站基本信息的设置，如网站名称、Logo、底部信息等  
## 布局：  
前台URL布局  
- /index.html : 站点首页  
- /news,reports,sugars,course-resources/index.html: 文章列表页  
- /news,reports,sugars,course-resources/show-[id].html: 文章详细页，[id]指各篇文章的唯一标识，暂用不重复的数字代替  
- /about.html : 关于作者页  
- /help.html : 站点帮助页  
### 源文件布局  
/      放置index、about、help等页面文件  
/posts/      放置前台相关的文章页面文件  
/admin/posts/      放置后台相关的文章页面文件  
/assets/      存放站点中所有的图片、文档等资源文件  
/css/     存放站点中所有的css文件  
/js/      存放站点中所有的前端脚本文件
